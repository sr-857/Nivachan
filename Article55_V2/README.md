# article55
Create a fair electoral system to select the President

Congratulations !! You have been chosen to design a error proof voting system for a gated Society.

Candidate configuration - 
You have to design an app where - 
1. Candidate can write there fullname.
2. Candidates can write there brief summary in 300 words.
3. Candidate can upload there photo (max 10 MB).

User configuration - 
1. User has to provide there basic information like Name, Block number, Flat Number, Phone number, Email (optional).
2. After successfully entering details, Users will see the main page where the user can caste their vote to President, Secretary and Treasurer.
There can be two options for this - 
Option a> List all candidates under same catergory and ask user to choose any one person.
option b> Each candidate has one upvote and downvote button, selecting upvote for one candidate will result in downvote for rest in the same category.
3. Users can see the live voting count but not other user details.
4. Only 1 person per flat can vote, if another person tries to impersonates, new votes can be rejected by admin.

Admin configuration - 
1. Admin has the right to approve candidates and can delete any candidature.
2. Admin can delete any vote given by a user.
3. Admin can see which user has voted to which candidate in the admin dashboard.

Software and applications that can be used.
1. Flutter for frontend.
2. Firebase/Google sheets or any DB for backend.
